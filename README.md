
# open-prompt-wiki

Repositorio de prompts listos para usar, diseñados para acelerar y mejorar el desarrollo de aplicaciones con inteligencia artificial. Cada prompt está pensado para cubrir una fase clave del ciclo de vida de una app: arquitectura, construcción, debugging, revisión, refactorización, testing y documentación.
Basado en la GUIA PRÁCTICA 2026 de Big School: 7 Prompts para Programar Más Rápido

## Prompts disponibles

- **01-El Arquitecto**: Diseño de arquitectura técnica completa para proyectos.
- **02-El Constructor**: Implementación de funcionalidades con convenciones claras.
- **03-Debugger**: Diagnóstico y debugging metódico usando Chain of Thought.
- **04-CodeReview**: Revisión rigurosa de código (seguridad, rendimiento, calidad).
- **05-Refactoring**: Refactorización para mejorar rendimiento, legibilidad y escalabilidad.
- **06-Testing**: Generación de suites de tests automatizados con cobertura completa.
- **07-Documentacion**: Creación de documentación técnica, README, docstrings y guía de API.

Cada archivo contiene un prompt detallado, listo para copiar y adaptar según tu proyecto.
